This is a simple util I made for Undertale or Fangames of said Undertale
This is a fast thought I just had, and now I'm uploading this
This uses .ahk files to run shortcuts
the current shortcuts are:
F1 - Do frisk dancing (Used for things like Undertale Yellow where you cant normally frisk dance)
F2 - Stop the frisk dance mid way
F3 - Auto skip the text go brrrrrr
F4 - stop the F3 go brrrrrrrrrrrrrr
